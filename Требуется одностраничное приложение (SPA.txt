Требуется одностраничное приложение (SPA) разработанное на vue (приветствуется)\angular\react, включающее в себя следующие элементы: 
Главная страница 
Меню приложения с одним вложенным элементом 
элемент "button" без поведения 
элемент "checkbox" - возможность взаимодействовать (включать/выключать) 
элемент "combobox" с двумя опциями - возможность взаимодействовать (переключать между опциями) 
элемент "textfield" 
элемент "grid" из трех столбцов ("Number", "String", "Date"), отображающий синтетический набор данных вида {1, 'ABC', '2022-09-07'}. Грид должен включать возможность сортировки набора данных а также наложение фильтров. 

Приложение должно либо использовать framework типа bootstrap-vue (и аналоги для angular/react), либо иметь мастер-файл с классами CSS для всех элементов на странице для централизованного управления стилями.  

Расположение на странице свободное, требований к оформлению нет, но элементы должны визуально располагаться отдельно друг от друга. 

Возможно использование сторонних плагинов для создания элементов, но ссылок на внешние библиотеки быть не должно.  
Приложение должно уметь работать без обращения в интернет.
